背景

设置背景图片，可同时设置背景图片和背景颜色

设置背景图片的重复方式

background-repeat的可选值：repeat默认值，背景会沿着x轴和y轴双向重复；repeat-x沿着x轴重复；repeat-y沿着y轴重复；no-repeat背景图片不重复。

设置背景图片的位置

top、left、right、bottom、center必须同时指定两个值，否则第二个值默认为center

水平方向偏移量 垂直方向偏移量

设置背景的范围

背景图片偏移量原点的设置

背景图片尺寸设置

背景图片是否跟随元素移动

背景相关的简写属性

background-size必须写在background-position后边并使用/隔开

background-origin必须在background-clip前边

解决图片闪烁问题-雪碧图

可以将多个小图片同时统一保存在一个大图片中，然后通过调整background-position来显示响应的图片，这样图片就会同时加载到网页中，就可以有效的避免出现闪烁的问题。这个技术在网页中应用十分广泛，被称为CSS-Sprite,雪碧图

雪碧图的使用步骤

1、先确定要使用的图标

2、测量图标的大小

3、根据测量结果创建一个元素

4、将雪碧图设置为元素的背景图片

5、设置一个偏移量以显示正确的图片

雪碧图特点：一次性将多个图片加载进页面，降低请求的次数，加快访问速度，提升用户的体验

渐变

渐变是图片，需要通过background-image设置

线性渐变

linear-gradient()

background-image: linear-gradient(to left,red,yellow);

to left;to right,to top;to bottom默认是向下

deg表示度数，向右是90，向下是180，向左270，0和360都是向上

turn表示圈数，0.25turn向右

渐变可以指定多个颜色，默认是平均分配，也可以手动指定渐变的分布情况

可以平铺的线性渐变

background-image: repeating-linear-gradient(to right , red, yellow 50px)

径向渐变

radial-gradient()径向渐变（放射性效果）

默认情况下径向渐变的形状根据元素的形状来计算

表格

通过table标签创建一个表格

在table中使用tr表示表格中的一行，有几个tr就有几行，在tr中使用td表示一个单元格，有几个td就有几个单元格

colspan=”2”表示横向合并两个单元格

rowspan=”2”表示纵向合并两个单元格

可以将表格分成：thead tbody tfoot th可以替换头部的td，会有加粗居中的效果

表格的样式

border-spacing设置边框之间的距离

border-collapse设置边框的合并

如果表格中没有使用tbody而是直接使用tr，那么浏览器会自动创建一个tbody，并且将tr全部放到tbody中，此时tr不是table的子元素

默认情况下元素在td中垂直居中的，可以通过vertical-align来修改

表单简介

过渡

过渡(transition)

​ 通过过渡可以指定一个元素属性发生变化时的切换方式

​ 可以创建一些非常好的效果，提升用户的体验

transition-property: ; 指定要执行过渡的属性

多个属性间需用逗号隔开；如果所有属性都需要过渡，则使用all关键字；大部分属性都支持过渡效果,注意过度时必须是一个有效值向另一个有效值进行过渡

transition-duration: ; 指定过渡效果的持续时间

时间单位：s 和 ms 1s=1000ms

transition-timing-function: ; 过度的时序函数

​ 指定过渡的执行方式

​ 可选值

​ ease 默认值 慢速开始，先加速，在减速

​ linear 匀速运动

​ ease-in 加速运动

​ ease-out 减速运动

​ ease-in-out 先加速后减速

​ cubic-bezier() 来回指定时许函数,通过下面的网址查找数字

​ https://cubic-bezier.com

steps() 分布执行过渡效果

​ 可以设置第二个值

​ end 表示时间结束时执行过渡（默认值）

​ start 表示时间开始时执行过渡

transition-delay: ; 过渡效果的延迟

​ 等待一段时间后执行过渡

transition 可以同时设置过渡相关的所有属性，只有一个要求，如果

​ 要写延迟，则两个时间中的第一个时间是持续时间，第二个是延迟时间

transition: 2s margin-left 1s cubic-bezier(.33,1.09,.39,-0.33);

####动画

动画

​ 设置动画效果需要先设置一个关键帧，关键帧设置了动画执行每一个步骤

设置box2的动画

animation-name: 要对当前元素生效的关键帧的名字

animation-duration: 动画执行的时间

动画的延时

​ animation-delay: 2s;

animation-timing-function: ease-in-out;

animation-iteration-count 动画执行的次数*

​ 可选值：

​ 次数

​ infinite 无限执行

animation-direction

​ 指定动画运行的方向

​ 默认值 normal 从from到to运行，每次都是这样

​ reverse 从to向from运行 每次都是这样

​ alternate 从from向to运行，重复执行时动画反向执行

​ alternate-reverse 从to向from运行，重复执行时动画反向执行

animation-play-state

​ 设置动画的执行状态

​ 可选值：

​ running 默认值 动画执行

​ paused 动画暂停

animation-fill-mode: ;

​ 动画的填充模式

​ 可选值：

​ none 默认值 动画执行完毕元素回到原来位置

​ forwards 动画执行完毕元素停止在动画结束的位置

​ backwards 动画延时等待时，元素就会处于开始位置

​ both 结合了forwards和backwards

变形

变形是指通过css来改变元素的形状或者位置

​ 变形 不会影响到页面的布局

transform用来设置元素的变形效果

​ 平移：

​ translateX() 沿着X轴方向平移，正值向右

​ translateY() 沿着Y轴方向平移，正值向下

​ translateZ() 沿着Z轴方向平移

平移元素，百分比是相对于自身计算的

旋转

通过旋转可以使元素沿着x,y,z旋转指定的角度，从xyz的正方向看见去是顺时针旋转的

变形原点

transform-origin: 20px 20px;默认值为center

对元素缩放

transform: scale(2);双方向放大两倍

scaleX()水平方向缩放

scaleY()垂直方向缩放

溢出父元素后可以加overflow
