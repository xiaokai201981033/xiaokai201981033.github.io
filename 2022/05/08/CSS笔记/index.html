CSS笔记

CSS：层叠样式表，网页实际上是一个多层的结构，通过CSS可以分别为网页的每一个层来设置样式

使用CSS修改元素样式

  1. 在标签内部通过style属性设置元素样式(内联方式,行内方式)

    缺点：只能对一个标签生效，并且当样式发生变化时，修改不方便

    注意：开发时绝对不要使用内联样式

  2. 将样式编写到head中的style标签里(内部样式表)，然后通过CSS选择器来选中元素并为其设置各种样式，可同时为多个标签设置样式，并且修改时只用修改一处.但是不能跨页面使用。

  3. 可以将CSS样式编写到一个外部的CSS文件中，然后通过link标签来引入外部的CSS文件（外部样式表）可以对多个网页进行使用

style里面不是HTML语言了，得遵循CSS语言规则

/*注释*/CSS的注释

样式的继承，我们为一个元素设置的样式同时也会应用到其后代元素上，继承是发生在祖先和后代之间的，但是并不是所有的继承都会发生：比如背景相关的，布局相关的样式不会被继承

选择器的权重

​ 内联样式 1000

​ id选择器 100

​ 类和伪类选择器 10

​ 元素选择器 1

​ 通配选择器 0

​ 继承的样式没有优先级

比较优先级时，需要将所有的选择器的优先级进行相加计算，最后优先级越高，则越优先显示（分组选择器）是单独计算div,span,p{}单独计算

选择器的累加不会超过其最大的数量级

可以在某一个样式的后边添加！important，则此样式优先级最高，甚至超过内联样式

注意：开发中慎用

如果优先级计算后相同，此时则优先使用靠下的方式

文档流

文档流是网页中最底下的一层，是一个概念，我们创建的元素默认都是在文档流中进行排列

元素主要有两个状态：在文档流中和不在文档流中(脱离文档流)

块元素特点

  1. 块元素在页面会独占一行
  2. 默认宽度是父元素的全部（会把父元素撑满）
  3. 默认高度是被内容撑开（子元素）

行内元素特点

  1. 行内元素不会独占页面的一行，只占自身的大小
  2. 行内元素在页面中从左向右水平排列，如果一行之中不能装满，则元素会换到第二行继续从左向右排列
  3. 行内元素的默认高度和宽度都是被内容撑开

盒子模型（box model）

盒子大小是可见框大小，内容区加内边距加边框

内容区（content）

元素的所有子元素和文本内容都在内容区排列，width设置内容区的宽度，height设置内容区的高度

内边距（padding）

内边距的颜色是盒子的颜色

边框（border）

边框至少要设置三个样式：宽度（border-width）、颜色（border-color）、样式（border-style）

border-width写了4个值时，顺时针对应，3个值时，上、左右、下，2个值时，上下、左右 border-top-width、bottom、right、left

border-color和border-width一样

border-style solid实线、dotted点状虚线、dashed虚线、double双线

简写：border: 10px solid orange;空格隔开，顺序不影响

外边距（margin）

外边距可以设置为负值

水平方向布局

一个元素在其父元素中，水平布局需要满足以下等式

margin-left+border-left+padding-left+width+padding-right+border-right+margin-right=其父元素内容区宽度

如果等式不满足，则称为过渡约束，等式会自动调整

  • 如果七个值中没有auto的情况，浏览器会自动调整margin-right的值使等式成立

  • 如果有一个值设置为auto，则会自动调整auto那个值以使等式成立

  • 如果一个宽度和一个外边距设置为auto，则宽度会调整到最大，设置为auto的外边距会自动为0

  • 如果两个外边距和一个宽度设置为auto，则宽度最大，外边距全为0

  • 如果两个外边距设置为auto，宽度为固定值，则宽度在父元素中居中，外边距值相同

垂直方向布局

子元素在父元素中排列的，如果子元素的height超过父元素，会溢出，使用overflow来处理溢出的子元素

overflow: visible;默认值，子元素会从父元素中溢出，在父元素外部的位置显示

overflow: hidden;溢出的内容会被裁剪，不会显示

overflow: scroll;生成两个滚动条，通过滚动条来查看完整的内容

overflow: auto;根据需要生成滚动条

overflow-x:单独处理水平方向

overflow-y:单独处理垂直方向

overflow设置在父元素中

外边距的折叠

垂直外边距的折叠

相邻的垂直方向外边距会发生折叠

兄弟元素

​ 兄弟元素间相邻垂直外边距会取两者最大值（两者都是正值）

​ 如果一正一负，则取两者的和

​ 如果都是负数，取绝对值大的那个

父子元素

​ 父子元素间相邻外边框，子元素会传递给父元素（上外边距）

​ 父子外边距折叠会影响到页面的布局

行内元素的盒子模型

  • 行内元素不支持设置宽度和高度
  • 行内元素可以设置padding，但是垂直方向padding不会影响页面布局
  • 可以设置border，垂直方向不会影响页面布局
  • 可以设置margin，垂直方向不会影响页面布局
display用来设置元素的类型

可选值

​ inline 将元素设置为行内元素

​ block 将元素设置为块元素

​ inline-block 将元素设置为行内块元素，既可以设置宽度和高度，又不会独占一行

​ table 将元素设置为一个表格

​ none 元素不在页面中显示

visibility用来设置元素的显示状态

可选值

​ visibility 默认值，元素正常显示

​ hidden 元素在页面中隐藏，不显示，但是依然占据页面的位置

盒子的大小

box-sizing:用来设置盒子尺寸的计算方式（设置width和height的作用）

content-box:默认值，宽度和高度为内容区的大小

border-box:宽度和高度用来设置整个盒子可见框的大小

轮廓和圆角

outline: 10px red solid;设置轮廓线，不会影响可见框的大小，加上之后不会影响相邻元素的位置

阴影

box-shadow:用来设置元素的阴影效果，阴影不会影响页面布局

第一个值 ：水平偏移量 设置阴影的水平位置 正值向右移动，负值向左移动

第二个值： 垂直偏移量 设置阴影的垂直位置 正值向下移动 负值向左移动

第三个值： 阴影的模糊半径

第四个值： 阴影的颜色

圆角

border-radius:可以分别指定四个圆角：border-top-left-radius、border-top-right-radius、border-bottom-right-radius、border-bottom-left-radius

border-radius: 50%设置为圆形

border-radius: 20px / 40px;

浮动

folat: left; folat: right; float: none

元素设置浮动后，水平布局的等式不需要强制成立，并且元素会从文档流中脱离，不再占用文档流的位置，所以元素下边的还在文档流中的元素会自动上移

设置浮动以后元素会向父元素的左侧和右侧移动，浮动元素默认不会从父元素中移出，浮动元素向左或向右移动时，不会超过它前面的其他浮动元素；浮动元素不会超过它上边浮动的兄弟元素最多最多和它一样高

浮动可以让页面中的元素水平排列

脱离文档流的特点

块元素

1、块元素不再独占页面一行

2、脱离文档流后，块元素的高度和宽度默认都被内容撑开

行内元素

行内元素脱离文档流后变成块元素，特点和块元素一样

网页布局

高度塌陷

高度塌陷的问题

​ 在浮动的布局中,父元素的高度默认是被子元素撑开的,当子元素浮动后,其

​ 完全脱离文档流,子元素从文档流中脱离将会无法撑起父元素的高度,导致父元素的高度丢失

​ 父元素高度丢失以后,其下的元素会自动上移,导致页面布局混乱

BFC(block formatting context) 块级格式化环境

​ BFC是一个CSS中隐含的属性,可以为一个元素开启BFC

​ 开启BFC后该元素会变成一个独立的布局区域

​ 元素开启BFC后的元素的特点:

​ 开启BFC的元素不会被浮动元素所覆盖

​ 开启BFC的元素子元素和父元素外边距不会重叠

​ 开启BFC的元素可以包含浮动的子元素

​ 可以通过一些特殊的方式来开启元素的BFC：

​ 1、设置元素的浮动

​ 2、将元素设置为行内块元素

​ 3、将元素的overflow设置为一个非visible的值

​ 常用的方式为元素设置overflow: hidden 开启其BFC，以使其可以包含浮动元素

定位

定位是一种更高级的布局手段，通过定位可以将元素摆放到页面的任意位置

使用position属性来设置定位

可选值：

static默认值，元素是静止的没有开启定位

relative开启元素的相对定位，开启之后需要设置偏移量，偏移是相对于元素在文档流中的位置，相对定位会提升元素的层级，但是没有脱离文档流，相对定位不会改变元素的性质

偏移量（offset）可以通过偏移量改变元素的位置 top定位元素与定位位置上边的距离、bottom、left、right

absolute开启元素的绝对定位,开启绝对定位后，如果不设置偏移量元素的位置不会发生变化；元素会从文档流中脱离；绝对定位会改变元素的性质，行内变成块，块的宽高被内容撑开；绝对定位会提升元素的层级；绝对定位元素是相对于其包含块进行定位的；

包含块（containing block）

​ 正常情况下包含块：是离当前元素最近的祖先块元素；

​ 绝对定位的包含块：离它最近的开启了定位的祖先元素；都没开启定位则根元素就是它的包含块

fixed开启元素的固定定位（也是一种绝对定位） 固定定位永远参照于浏览器的视口进行定位，可视化的网页窗口，不会随网页的滚动条滚动；其他特点与绝对定位相同

sticky开启元素的粘滞定位 与相对定位的特点基本一致，不同的是粘滞定位可以在元素到达某个位置时将其固定

绝对定位元素的位置

开启绝对定位后水平布局：left+margin-left/right+border-left/right+padding-left/right+width=包含块宽度

垂直布局：top+margin-top/bottom+border-top/bottom+padding-top/bottom+height=包含块高度

可设置auto的值有 margin width left right

left和right默认值是auto，
